# Instruction Memory (Verilog)
# Overview

The Instruction Memory is a key component in a processor that stores the program instructions.
It provides the instruction corresponding to the address generated by the Program Counter (PC).

This module implements a 16 × 8-bit instruction memory using Verilog HDL.

# Features
 Stores 16 instructions (memory locations)
 Each instruction is 8 bits wide
 Combinational read (no clock required)
 Address-driven instruction fetch

# Inputs & Outputs
Signal	Type	Description
addr	Input [3:0]	Address from Program Counter
instruction	Output [7:0]	Instruction at given address

# Functional Behavior
addr changes → instruction updates instantly
No clock required
Pure combinational logic

# Key Concepts Used
Memory array declaration:
reg [7:0] memory [0:15];
Combinational read:
always @(*)

# Applications
 Instruction Fetch stage in CPUs
 Program storage (ROM)
 Control unit input
 RISC processor design
